Skill : Html5, Css3, Javascript
Usage libraies and API : Git, Firebase, fontawesome, jquery

html5 와 css3를 배우고 처음으로 만든 클론코딩이다. 아직 익숙하지 않은 html5 와 css3로 여기어때 사이트를 따라 만들어 보면서 프로그래밍에 조금이라도 더 적응하기위해 만들었다. 프로젝트를 만들면서 써보지 않았던 미디어 쿼리 자바스크립트를 사용하여 어려움이 있었으나 차근차근 필요한 기능이 무엇인지 생각하고 만들어보았다 미디어 쿼리를 이용하여 반응형 웹사이트를 만드는 데 태블릿, 모바일로 만들면서 화면에 맞는 크기 조절을 하는데 어려움을 느꼈다 이를 크롬 확장 프로그램을 이용하여 각각의 화면을 보면서 만들어보니 구성이 어떻게 되는지 상상이 되어서 만들 수 있었다 또 자바스크립트를 이용하여 목차를 만들어 보았는데 생각보다 간단한 자바스크립트여서 생각 이외로 쉽게 만들었다 이를 만들어보면서 상황별로 어떤거를 써야하는지 생각이 들기 시작했고 상황에따라 사용할수 있는 것도 많다고 느꼈다 css를 하면서 완전한 정답이 있지는 않다고 생각이 들었지만 더욱 간단하고 이쁘게 만드는 방법은 있는 것 같다. 가능한 내가 보고있는 화면과 비슷하게 만들기 위해 노력했다.
